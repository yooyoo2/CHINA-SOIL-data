# 中国土壤属性数据集（CSDLv2）分析项目

## 1. 项目简介

本项目基于中国陆面过程模拟土壤属性数据集（CSDLv2），对土壤理化性质进行整理与分析，并探讨土壤属性与农业生态系统生产力之间的关系。该数据集可为土壤养分研究、作物生产模拟和土地利用评价提供基础数据支持。

---

## 2. 数据集说明

本项目使用的数据来源于：

**A China dataset of soil properties for land surface modelling (version 2, CSDLv2)**

CSDLv2 是面向陆面过程模型构建的中国土壤属性空间数据集，整合了全国范围内多来源土壤调查资料，形成连续栅格化土壤信息。

### 数据特点

* 空间分辨率：1 km × 1 km
* 覆盖范围：中国区域
* 土层深度：多层土壤剖面
* 数据格式：栅格数据

### 主要变量

* 土壤有机碳（SOC）
* 全氮（TN）
* pH
* 容重（Bulk density）
* 砂粒、粉粒、黏粒含量
* 土壤水力学参数

## 8. 项目结论
本项目基于CSDLv2数据集，完成了中国区域土壤属性数据的预处理、空间分布特征分析与可视化，验证了该数据集能够较好地反映我国土壤属性的空间异质性与分布规律。
研究结果表明，CSDLv2数据集能够较好地反映中国区域土壤属性空间分布特征，可为后续农业生态系统生产力评估、陆面模型参数化及相关区域研究提供可靠的数据基础与参考依据。
